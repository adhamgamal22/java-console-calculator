# 🧮 Java Console Calculator

A simple and clean **console-based calculator** built using Java with a focus on **clean code, OOP design, and error handling**.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* ❌ Division by zero handling
* ⚠️ Input validation (prevents crashes)
* 🔁 Interactive loop (run multiple operations)

---

## 🧠 Concepts Used

* Object-Oriented Programming (OOP)
* Enums for operations
* Exception Handling
* Clean Code Principles
* Separation of Concerns


---

## 💡 Example

```
Select your Operation:
1. Addition (+)
2. Subtraction (-)
3. Multiplication (*)
4. Division (/)
5. Exit

Enter first number: 10
Enter second number: 5

Result = 15
```
